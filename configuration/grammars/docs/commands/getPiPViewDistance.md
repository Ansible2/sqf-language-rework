Get the set PiP (Picture-in-Picture) view distance (that can be set by `setPiPViewDistance`).


---
*Syntaxes:*

`getPiPViewDistance`

---
*Example 1:*

```sqf
private _pipViewDistance = getPiPViewDistance;
```

*Example 2:*

```sqf
private _pipViewDistanceRatio = getPiPViewDistance / viewDistance;
```
