Enable / disable `Post Process Effects`


---
*Syntaxes:*

effect `ppEffectEnable` enable

effect `ppEffectEnable` enable

effectArray `ppEffectEnable` enable

---
*Example 1:*

```sqf
"colorCorrections" ppEffectEnable true;
```

*Example 2:*

```sqf
_hndl ppEffectEnable true;
```

*Example 3:*

```sqf
[_hndl1, _hndl2] ppEffectEnable true;
```
