Returns an array of all currently placed `Eden Editor entities`.


---
*Syntaxes:*

[[all3DENEntities]]

---
*Example 1:*

```sqf
systemChat str all3DENEntities;
```

*Example 2:*

```sqf
all3DENEntities params ["_objects", "_groups", "_triggers", "_systems", "_waypoints", "_markers", "_layers", "_comments"];
```
