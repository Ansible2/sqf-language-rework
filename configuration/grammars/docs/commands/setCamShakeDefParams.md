Sets camera shake default parameters.


---
*Syntaxes:*

`setCamShakeDefParams` [power, duration, frequency, minSpeed, minMass, caliberCoefHit, vehicleCoef]

---
*Example 1:*

```sqf
setCamShakeDefParams [1, 2, 3, 4, 5, .5, .5];
```

*Example 2:*

```sqf
setCamShakeDefParams [100, 10, 10, 4, 5, 1, 1];
```
