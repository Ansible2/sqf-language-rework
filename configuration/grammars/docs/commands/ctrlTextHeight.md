Returns the control text height. Supported control `types` are:
* `CT_STATIC`
* `CT_STRUCTURED_TEXT`
* `CT_EDIT` 
Since Arma 3 v2.05.147765 control types added:
* `CT_XKEYDESC`
* `CT_XBUTTON`
* `CT_SHORTCUTBUTTON`
* `CT_BUTTON`
* `CT_ACTIVETEXT`


---
*Example 1:*
```sqf
_h = ctrlTextHeight _ctrl;
```
