#### Description:
Gets an object or group from a netId.

#### Parameters:
0: **_id** *(STRING)* - The id of the object

1: **_fromObject** *(BOOL)* - false for getting object, true for group

#### Returns:
<OBJECT or GROUP> - The Id of the entity

#### Examples:
```sqf
_entity = ["0:0"] call KISKA_fnc_netId;
```
