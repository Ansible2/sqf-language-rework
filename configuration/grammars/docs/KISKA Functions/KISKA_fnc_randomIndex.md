#### Description:
Returns a random index of an array ~2x faster than BIS_fnc_randomIndex;

#### Parameters:
0: **_array** *(ARRAY)* - The array to find a random index of.

#### Returns:
<NUMBER> - The random index

#### Examples:
```sqf
private _randomIndex = [[1,2,3]] call KISKA_fnc_randomIndex;
```
