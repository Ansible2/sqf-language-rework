#### Parameters:
0: **_baseConfig** *(CONFIG)* - The config path of the base config

#### Returns:
<HASHMAP> - see KISKA_fnc_bases_getHashmap

#### Examples:
```sqf

    "SomeBaseConfig"
] call KISKA_fnc_bases_createFromConfig_simples;
```
